What is the Document Object Model (DOM)?

DOM is the structure of a website which contain nodes in tree structure

How do you select an element from the DOM?

There are several ways of select element from the DOM. We select element by its tag name. We also select elements by 'id' name, 'class' name, and by query selector.
such as:-
1. getElementsByTagName
2. getElementsByClassName
3. getElementById
4. querySelector
5. querySelectorAll

Event delegation is the way of using one Event handlar instead of using multiple event.It gives the power of event to parent rethar than child
